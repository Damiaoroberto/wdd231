Individual Project Site Plan

Site Name

Photographer - Damiao Machote

This name represents a personal photography portfolio website for Damiao Machote, showcasing his work and services.

Optional domain availability: PhotographerDM.com

Site Purpose

The site will serve as an online portfolio for Damiao Machote's photography work. It will include galleries, a biography, client testimonials, and a contact form for inquiries.

Scenarios

  • How can I view the photographer's portfolio?
  • How do I contact Damiao Machote for a photography session?

Color Schema

Primary Color: #2E6B8B (Dark Blue) - Used for headings and call-to-action buttons.

Accent Color: #FF6347 (Tomato Red) - Used for emphasis, links, and important notifications.

Typography

Primary Font: Arial, sans-serif - Used for all body text for clarity and readability.

Secondary Font: 'Georgia', serif - Used for headings to create a formal contrast with the body text.
